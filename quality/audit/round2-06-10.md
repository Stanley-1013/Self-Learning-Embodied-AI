## Ch07 — Forward Kinematics / DH

**New findings: 1 (LOW)**

### L-07-01 (LOW): "Standard vs Modified DH" axis-alignment rule — precision check

- **Location**: Line 79 — "Standard DH: frame i's $Z_i$ 對齊**第 $i+1$ 軸**"
- **Category**: 術語混用 / 事實
- **Issue**: The claim "Standard DH $Z_i$ aligns with axis $i+1$" is the Craig-book paraphrase of Denavit-Hartenberg. The original DH 1955 convention is actually stated as: "axis $i$ of the joint is denoted $z_{i-1}$" (i.e., joint axis $i$ lives on frame $i-1$). This is equivalent to the textbook's statement but often stated the other way:
  - Craig's Modified DH: $Z_i$ along joint axis $i$ → frame $i$'s $Z$ = axis $i$ ✓ (text correct)
  - Standard (classical) DH: $Z_i$ along joint axis $i+1$ → frame $i$'s $Z$ = axis $i+1$ ✓ (text correct)
- **Verdict**: Statement is correct and matches Craig's Ch3 boundary definition. However the 乘法順序 claim (Standard: `Rot_Z → Trans_Z → Trans_X → Rot_X`; Modified: `Rot_X → Trans_X → Rot_Z → Trans_Z`) is slightly ambiguous because implementations sometimes group these differently. For Craig's Modified DH, the canonical form is ${}^{i-1}T_i = R_X(\alpha_{i-1}) \cdot T_X(a_{i-1}) \cdot R_Z(\theta_i) \cdot T_Z(d_i)$ — matches text. No change needed.
- **Action**: None required. Low-priority precision note.

## Ch09 — Robot Dynamics Modeling

**New findings: 2 (MEDIUM, LOW)**

### M-09-01 (MEDIUM): RNEA "frame $i$" convention note self-contradicts the formulas

- **Location**: Line 117 (convention note) vs lines 120–127 (forward pass formulas)
- **Category**: 術語混用 / 推理斷鏈
- **Round-1 fix**: The convention block was added to disambiguate frames.
- **Issue**: The note says 「以下**所有**向量均表達在 frame $i$ 下」. But the formula $\omega_i = R_i^T(\omega_{i-1} + \dot q_i \hat z)$ only makes sense if $\omega_{i-1}$ is expressed in **frame $i-1$** (its own frame), and $R_i^T$ rotates it to frame $i$. If "all vectors are in frame $i$" literally, then multiplying $\omega_{i-1}$ by $R_i^T$ is meaningless. Same issue in the backward pass: $R_{i+1} f_{i+1}$ only works if $f_{i+1}$ is in frame $i+1$ and $R_{i+1}$ maps it back to frame $i$.
- **Correct statement** (Craig convention): "Each indexed variable ($\omega_i$, $f_i$, $n_i$, $r_i$, etc.) is expressed **in its own frame $i$**. $R_i$ denotes the rotation matrix $^{i-1}R_i$ that maps frame $i$ vectors into frame $i-1$ coordinates; $R_i^T = {^i R_{i-1}}$ performs the inverse."
- **Suggested fix**: Replace the blanket "all vectors in frame $i$" with a per-variable convention statement. The round-1 fix addressed the right concern but introduced a logical inconsistency.

### L-09-02 (LOW): Pseudo-inertia LMI attribution

- **Location**: Line 471 — "等價於 Souloumiac / Traversaro 的 pseudo-inertia matrix"
- **Category**: 事實
- **Issue**: The standard references for the pseudo-inertia LMI (physical realizability via PSD pseudo-inertia matrix) are **Traversaro, Prete, Nori 2016** and **Wensing, Kim, Slotine 2017**. "Souloumiac" is a well-known name in ICA / Jacobi joint diagonalization, not in robot inertia identification. This looks like a misattribution.
- **Suggested fix**: Replace "Souloumiac / Traversaro" with "Traversaro 2016 / Wensing-Kim-Slotine 2017" or simply "pseudo-inertia LMI (Traversaro 2016)".

### Round-1 HIGH fix verification

**LMI triangle inequality (Ch09 line 471):**
- Text states: "$I_{jj} + I_{kk} \geq I_{ii}$ 對所有 $\{i,j,k\}$ 的排列都成立"
- Correct: triangle inequality on principal inertia = 3 distinct inequalities (permuting which axis is "$i$"). Saying "for all permutations of $\{i,j,k\}$" covers all 3 cases, and the attribution as "equivalent to pseudo-inertia PSD LMI" is mathematically correct (both capture physical realizability: positive mass + positive-definite inertia + triangle inequality). **Statement is complete and correct.** Minor phrasing could be tightened to "for each of the 3 cyclic choices of $(i,j,k)$" but not required.

**Slotine-Li / Barbalat (Ch09 line 507):**
- Text states: "因 $V$ 有下界且 $\dot V \leq 0$，故 $s \in L^\infty \cap L^2$；再由 Barbalat's Lemma（$\dot V$ 均勻連續）推得 $s \to 0$"
- Check: The standard Barbalat's lemma needs $\dot V$ uniformly continuous (sufficient via bounded $\ddot V$). The chain $V$ bounded + $\dot V \leq 0$ ⇒ $V \to V_\infty$; then UC of $\dot V$ ⇒ $\dot V \to 0$ ⇒ $s^T K s \to 0$ ⇒ $s \to 0$. **Proof sketch is correct; uniform continuity requirement is explicitly stated.** ✓
