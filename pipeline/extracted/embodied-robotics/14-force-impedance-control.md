## 定義
- 位置控制 vs 力控制：幾何追蹤(剛性) vs 接觸力調節(柔順)；搬運vs裝配
- Impedance：F=Md*ẍ+Dd*ẋ+Kd*(x-xd)；輸入位移→輸出力；適合直驅/柔性馬達
- Admittance：對偶；輸入力→輸出位移修正；適合剛性大慣量工業臂
- Hybrid position/force：Selection Matrix 分解方向（法向力控+切向位控）
- 感測：直驅電流推算(頻寬高/摩擦干擾) / 關節力矩感測器(全身柔順/碰撞偵測) / F/T sensor(六維末端/需重力補償)

## 閉環
- 控制末端，替代/搭配位置控制
- 依賴 Ch09 動力學前饋(重力/慣性補償) + Ch13 PID 關節跟蹤
- 一句話：「力控是賦予機器人觸覺反射的神經樞紐」

## 直覺 + 誤解
- 阻抗=拉彈力繩握方向盤、導納=推超市推車
- 4 誤解：精密裝配光靠位置就好(零件公差)、阻抗=導納(因果反)、F/T 直接讀=接觸力(需動態補償)、剛度調高就追準(高頻震盪)

## 3 情境題
1. 研磨恆定10N → 力位混合(法向PI力控+切向位控) + 重力補償 + 自適應剛度
2. 協作手引示教零力感 → 導納控制 K=0 B=小 + 精確重力/摩擦補償
3. 位控切力控衝擊 → 狀態機(接近→初接觸→穩態) + 接觸前降剛度提阻尼
